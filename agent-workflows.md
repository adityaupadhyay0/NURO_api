# Agent Workflows: NeuroMark Pro 10x

## Campaign Optimization Workflow
1. **User Goal:** User enters a goal (e.g., "Optimize my ad for Gen Z").
2. **Neuro-Analyst:** Processes raw neuro-metrics and identifies "Attention Gaps".
3. **Creative Strategist:** Receives gaps and develops specific hooks and creative angles.
4. **Media Buyer:** Suggests platform-specific funnel structures based on the strategy.
5. **Orchestrator:** Summarizes agent outputs into a unified marketing plan for the user.

## Synthetic Persona Weighting Workflow
1. **Persona Entry:** User describes a specific niche persona (e.g., "Skeptical luxury watch enthusiast").
2. **Audience Agent:** Analyzes the persona's psychographics and generates a set of ROI multipliers (Attention, Emotion, Reward, etc.).
3. **NeuroEngine:** Applies these dynamic multipliers to the raw TRIBE v2 activations to calibrate marketing KPIs specifically for that persona.
4. **Calibration:** The system provides a "Persona Match Score" reflected in the Winning Probability.
